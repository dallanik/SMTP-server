Проект "SMTP сервер". Позволяет отправлять сообщения на указанную авторизованную почту через тот или иной почтовый сервер.

Составляющие проекта:
1)smtp.py. Файл с кодом, реализует сам сервер и отправку сообщений.
2)msg.txt. Файл с текстом сообщения, которое будет отправлено.
3)icon.png. Приклепляемое к сообщению изображение.

Возможные конфигурации:
1)Почта, на которое будет отправлено письмо. Для конфигурации необходимо поменять соответствующие значения логина и пароля в файле smtp.py.
2)Почтовый сервис, используемый для отправки сообщений. Для конфигурации необходимо поменять соответствующие значения наименования сервиса и порта в файле smtp.py.
3)Текст отправялемого сообщения. Для конфигурации изменить сдержание файла msg.txt.
4)Прикрепляемое к сообщению изображение. Для конфигурации изменить \ заменить изображение icon.png.
